# Project: Smart Farm Marketplace E-commerce

> ระบบร้านค้าส่วนกลาง: แอดมินเป็นคนลงสินค้าทั้งหมด

---

## 👥 [EPIC-01] ระบบจัดการผู้ใช้งาน (User & Authentication)

### [ ] Task 01.1: หน้าสมัครสมาชิก (Register)

**Requirement:** รองรับการกรอกข้อมูล Username, Password, ชื่อ-นามสกุล, ที่อยู่จัดส่ง และเบอร์โทรศัพท์

---

### [ ] Task 01.2: หน้าเข้าสู่ระบบ (Login) — ฝั่ง User

**Requirement:** ระบบตรวจสอบ Username และ Password เพื่อเข้าใช้งานหน้าหลัก

---

### [ ] Task 01.3: หน้าข้อมูลส่วนตัว (Profile Management)

**Requirement:** หน้าแสดงข้อมูลผู้ใช้, สามารถกดแก้ไขชื่อ เบอร์โทร และเพิ่ม/เปลี่ยนที่อยู่จัดส่งได้ รวมถึงมีปุ่มออกจากระบบ (Logout)

---

## 🛒 [EPIC-02] หน้าบ้านสำหรับลูกค้า (User Frontend / Shopping)

### [ ] Task 02.1: หน้าแรกของระบบ (Home Page Layout)

**Requirement:** จัดทำโครงสร้างหน้าแรก ประกอบด้วยส่วนของ Banner โปรโมชัน และโซนแสดงสินค้าขายดี

---

### [ ] Task 02.2: แถบค้นหาและเมนูนำทาง (Search & Navigation Bar)

**Requirement:** ทำแถบค้นหาสินค้า และเมนูแยกหมวดหมู่หลัก (ผัก / ผลไม้)

---

### [ ] Task 02.3: ระบบเมนูย่อย (Category Dropdown)

**Requirement:** เมื่อคลิกหรือเอาเมาส์วางที่หมวดหมู่หลัก จะมี Dropdown แสดงประเภทสินค้าย่อยลงมา

---

### [ ] Task 02.4: หน้าแสดงรายการสินค้า (Product Grid Display)

**Requirement:** แสดงบล็อกสินค้า (การ์ดสินค้า) เรียงเป็นแถว โดยดึงรูปภาพ, ชื่อสินค้า, และราคามาแสดงผล

---

### [ ] Task 02.5: ปุ่มเพิ่มสินค้าลงตะกร้า (Add to Cart Action)

**Requirement:** พัฒนาปุ่มเครื่องหมายบวก (+) บนการ์ดสินค้า เพื่อให้กดเลือกสินค้าเข้าตะกร้าได้

---

## ⚙️ [EPIC-03] หลังบ้านสำหรับผู้ดูแลระบบ (Admin Dashboard & Management)

### [ ] Task 03.1: หน้าเข้าสู่ระบบสำหรับแอดมิน (Admin Login)

**Requirement:** หน้าล็อกอินแยกเฉพาะสำหรับ Admin ป้องกันไม่ให้ผู้ใช้ทั่วไปเข้าถึงหลังบ้าน

---

### [ ] Task 03.2: ตารางแสดงรายการสินค้าหลังบ้าน (Product List View)

**Requirement:** หน้าแสดงรายการสินค้าทั้งหมดที่อยู่ในระบบในรูปแบบตาราง เพื่อให้แอดมินตรวจสอบสถานะ สต็อก และราคาได้ง่าย

---

### [ ] Task 03.3: ฟอร์มเพิ่มและแก้ไขข้อมูลสินค้า (Product Form Wizard)

**Requirement:** พัฒนาระบบฟอร์มให้แอดมินกรอกข้อมูลเพื่อบันทึกลงฐานข้อมูล โดยต้องมีช่องกรอกดังนี้:

- [ ] รหัสสินค้า (Product ID)
- [ ] ประเภทสินค้า (หมวดหมู่/ประเภทย่อย)
- [ ] ระบบอัปโหลดรูปภาพสินค้า (Product Image)
- [ ] ชื่อสินค้าและรายละเอียด (Product Name & Description)
- [ ] จำนวนสินค้าในสต็อก (Stock Quantity)
- [ ] ราคาสินค้า (Price)
- [ ] [แนะนำเพิ่ม] ชื่อเกษตรกร/ฟาร์มต้นทาง (Farmer/Farm Source)

---

## 📦 [EPIC-04] [แนะนำให้ทำเพิ่ม] ระบบจัดการคำสั่งซื้อ (Order Management)

> เนื่องจากแอดมินเป็นคนคุมระบบกลาง จำเป็นต้องมีส่วนนี้เพื่อดูว่าลูกค้าสั่งอะไรเข้ามา

### [ ] Task 04.1: ระบบจัดการออเดอร์หลังบ้าน (Admin Order View)

**Requirement:** หน้าจอสำหรับแอดมินดูรายการสั่งซื้อที่เข้ามาจากลูกค้า แสดงชื่อผู้ซื้อ, รายการสินค้าที่สั่ง, ยอดเงิน และที่อยู่จัดส่ง

---

### [ ] Task 04.2: สถานะการจัดส่ง (Order Tracking Status)

**Requirement:** แอดมินสามารถอัปเดตสถานะออเดอร์ได้ (เช่น: รอตรวจสอบ ➡️ กำลังแพ็ค ➡️ จัดส่งแล้ว)
